# sassy

Learning SASS
- benefits of SASS
-- you can nested styles! for example, instead of:
    header {
        ...
    }

    header button {
        ....
    }
    you can have:
    header {
        ...
        button {
            ...
        }
    }
-- you can use mixins similar to a function in JS
-- can extend created styles and add on/overwrite
-- do calculations with - + / *
